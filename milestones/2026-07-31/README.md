# July 31 Prototype

Goal

Produce the first playable RealMe prototype.

The objective is not feature completeness.

The objective is to validate the core interaction model.

---

Approved reference screens

- Home
- Today
- Map

These are canonical visual references.

Future iterations may improve details but should preserve the same design language and interaction philosophy.

---

Core principles

- Home reduces cognitive load.
- Today answers "What deserves my attention?"
- Map answers "Where am I in my life?"
- Living Input is available from every screen.
