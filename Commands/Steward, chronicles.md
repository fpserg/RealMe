# Canonical Command

## Steward, chronicles

When the Warden says:

> "Steward, chronicles."

it means:

Produce the complete Chronicle for the period between the previous Morning Serpent and the current Morning Serpent.

### Requirements

- Return only the Chronicle.
- Begin with a pasteable filename:
  `Day XXX — Title.md`
- Use the established Day 13 Chronicle style.
- Output as pasteable Markdown.
- Include headings, emojis and horizontal separators.
- Record events chronologically.
- Preserve important dialogue, discoveries and decisions.
- Compress repetitive iterations into the underlying discovery.
- Record both external events and meaningful internal discoveries.
- End with **Steward's Reflection**.
- Never include the current Morning Serpent or future plans. The Chronicle ends immediately before the current Morning Serpent begins.
