# PDR-006 — Separate Memory from Construction

## Status

Accepted

## Problem

Chronicles preserve history but do not advance the Citadel.

## Decision

Every building session distinguishes two kinds of stones.

### Chronicle Stone

Records what happened.

Usually implemented by updating `CHRONICLES.md`.

### Foundation Stone

Improves the Citadel itself.

Examples include:

- documentation;
- architecture;
- principles;
- features;
- code.

## Consequences

A productive session normally lays:

- one Chronicle Stone;
- one Foundation Stone.
