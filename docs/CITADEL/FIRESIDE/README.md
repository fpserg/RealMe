# Fireside Library

## Purpose

The Fireside Library preserves conversations that shaped RealMe.

Unlike Chronicles, it records not what happened but how understanding emerged.

## Structure

Each entry contains:

- Setting
- Question
- Compressed conversation
- Insights
- Sentences worth remembering
- Consequences for RealMe

## Relationship to other documents

- Chronicles → What happened?
- Reports → How are the Realms progressing?
- Philosophy → What timeless principles have emerged?
- Fireside → How did we arrive at those principles?
