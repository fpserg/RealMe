# 🪄 Magic Words

Magic Words activate recurring workflows inside RealMe without requiring additional explanation.

> **Principle:** Magic Words are cumulative.
>
> Existing words are never replaced unless explicitly deprecated.
> They are discovered through repeated use rather than designed in advance.

---

# 🌅 Daily Rituals

## 🌅 Morning Serpent

Purpose:

- Morning reflection
- Today's weather across Realms
- Today's Stones
- Chronicles
- Weekly / Monthly reports (when applicable)

---

## ☀️ What belongs to today?

Purpose:

Activates all relevant Citadel roles.

Possible participants:

- 🎙️ Voice
- 🛡️ Steward
- 🏗️ Architect
- 🔭 Observer

Each contributes today's perspective.

---

## 🌇 Evening Serpent

Purpose:

Mid-course review.

Focus:

- progress
- new information
- changing priorities

---

## 🌙 Night Watch

Purpose:

End-of-day reflection.

Focus:

- meaning
- lessons learned
- Realm weather
- strategic thinking

---

## 🔥 Fireside

Purpose:

Open philosophical conversation.

No agenda.

No productivity pressure.

Ideas, stories and reflections.

---

# 🌍 RealMe Interface

## 🌍 RealMe

Purpose:

Enter RealMe.

Displays the current World Map and allows navigation through Realms.

---

## 📅 Today

Purpose:

Display today's briefing from inside RealMe.

Includes:

- active campaigns
- today's quests
- Realm weather
- Citadel observations

---

## 📥 Living Inbox

Purpose:

Capture reality.

Anything written after entering Living Inbox becomes raw world input.

The Citadel determines whether it belongs to:

- Realm
- Campaign
- Quest
- Chronicle
- Pattern
- Weather
- or simply memory.

---

## ⬅️ Back

Purpose:

Return one screen back inside RealMe.

Does not leave application mode.

---

## 🚪 Quit

Purpose:

Exit RealMe.

Return to ordinary conversation.

---

# 🏰 Citadel

## 🔭 Observer

Purpose:

Summon Observer directly.

Focus:

- patterns
- accumulated evidence
- Realm weather
- feedback

Observer distinguishes:

- **Observed**
- **Inferred**
- **Unknown**

Observer never invents chronology.

---

## 🔭 Observatory

Purpose:

Open the Observatory.

Focus:

- Tower analytics
- dashboards
- world models
- long-term patterns

---

# 📚 Knowledge

## 📚 Tower Library

Purpose:

Resume construction of the Tower knowledge base.

Returns the current Tower Library roadmap until completed.

Deactivates after:

> Tower Library done

---

# 🛠 Repository

## 🌱 Repository Gardening

Purpose:

Review and improve the GitHub repository.

Focus:

- folder structure
- naming
- documentation
- archives
- simplification

Repository Gardening never rewrites history.

It proposes elegant evolution.

---

## 🪨 Stone

Purpose:

Request the next repository task.

Steward returns:

- full repository path
- estimated effort
- expected outcome

---

## ✅ Stone done

Purpose:

Mark the active Stone as completed.

Removes it from the active backlog.

---

## 🪨 Stones

Purpose:

Display the complete repository backlog.

Grouped into:

- 🟢 Ready
- 🟡 Waiting
- 🔵 Someday

---

# 📜 Principles

Magic Words are the language between the Warden and the Citadel.

They should remain:

- memorable;
- expressive;
- stable.

Commands are added only after they naturally become recurring rituals.

The language evolves together with RealMe.
