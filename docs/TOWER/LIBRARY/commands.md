# Tower Commands

## /check_coin ALL

Purpose:

Returns total balances held in player Treasuries.

Used for:

- Treasury liabilities
- Liquidity analysis

---

## /check_coin STAKE

Purpose:

Returns total balances currently staked.

Used for:

- Dividend snapshots
- Staking participation

---

## /check_coin DIV

Purpose:

Returns dividend balances awaiting transfer to Treasury.

Used for:

- Outstanding dividend liabilities

---
