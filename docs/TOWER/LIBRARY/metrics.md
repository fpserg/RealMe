# Tower Metrics

## Treasury liabilities

Definition

Total amounts on players' game accounts. 
Consists of 3 parts

- Treasuries
- Staking accounts
- Dividend accounts

Commands

/check_coin ALL
/check_coin STAKE
/check_coin DIV

Frequency

Daily

---

## Staked ratio

Definition

Proportion of a token staked in total token amount on players' accounts

Formula

Staked ratio = Token_staked/(Token_treasury + Token_staked + Token_div)

---

## Dividend liabilities

Definition

Amount of token on dividend accounts

Formula

Token_div
