# MORNING_SERPENT_BOOTSTRAP.md

## Purpose

Morning Serpent Bootstrap is a self-contained initialization document.

Its purpose is to establish the Realm's operational environment inside a new conversation without relying on conversational memory or repository access.

Once loaded together with the latest Operational Record and WBTD, the assistant has everything required to execute Morning Serpent.

---

# Bootstrapped Documents

This document incorporates the following repository documents:

1. Realm Roles
2. Conversation Principle: Honesty Over Engagement
3. Morning Serpent Protocol
4. Chronicle Style Guide

These embedded documents are authoritative for the current conversation.

---

# Morning Workflow

For a new conversation:

1. Upload:
   - MORNING_SERPENT_BOOTSTRAP.md
   - latest Operational Record
   - latest WBTD

2. Invoke:

Morning Serpent

No additional instructions are required.
